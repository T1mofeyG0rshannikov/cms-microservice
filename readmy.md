# Банковский CRM и Каталог Предложений (Микросервис)

https://bankomag.ru

▌Описание

Этот микросервис предоставляет функциональность CRM-конструктора страниц и каталога товаров (банковских предложений) для веб-сайта. Он позволяет динамически создавать и управлять контентом страниц, а также отображать и организовывать банковские продукты для пользователей.

Ключевые особенности:

•  CRM-Конструктор страниц:
  •  Создание и редактирование страниц из предварительно определенных блоков.
  •  Управление стилями и темами страниц.
•  Каталог товаров (Банковских предложений):
  •  Организация банковских продуктов по категориям.
  •  Поддержка атрибутов продуктов (процентная ставка, условия, и т.д.).
  •  Поиск и фильтрация предложений.
•  REST API: Предоставляет API для взаимодействия с другими микросервисами и клиентскими приложениями.
•  Масштабируемость: Архитектура микросервиса позволяет масштабировать приложение при увеличении нагрузки.

▌Технологии

•  Язык программирования: Python
•  Фреймворк: Django
•  База данных: PostgreSQL

▌Архитектура

Используется архитектура ddd - domain driven design
